# TransCL

TransCL is a multi-level automatic migration framework from CUDA to OpenCL. TransCL achieves automated migration (from source code to runtime) across four dimensions: source code transformation, programming model, memory model, and execution model, capable of supporting the migration of CUDA projects of varying scales. Leveraging OpenCL's portability, existing or newly developed CUDA projects can be migrated to diverse computing platforms.
